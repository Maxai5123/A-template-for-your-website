# Проект: "Шаблон для вашего сайта"

## Описание проекта
Готовый каркас для вашего сайта. В нём есть шапка и подвал, в котором вы можете разместить ссылки на какую-либо информацию

## Цели проекта
Создание шаблона для сайта с шапкой и подвалом

## Используемые технологии
- `HTML` для разметки и подключения функций сайта.
- `CSS` для придания красок сайту.
